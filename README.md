# Helpdesk Ticket Board (Next.js, JS + JSX) 

A Mini Helpdesk Board with filters, search, queue, and simulated live updates. 

## Run
clone repository 
open new terminal from repo folder
npm install
npm run dev
# open hhtp://localhost:3000

## Features
- [x] Next.js App Router structure & API route
- [x] Components with JSX + stable keys
- [x] Lifted state in Board (tickets, filters, search, queue)
- [x] Controlled inputs (status, priority, search)
- [x] Effects: fetch on mount; live updates with cleanup
- [x] Conditional UI: loading, error, empty
- [x] Queue sidebar: add/remove/clear
